Component anatomy
Interaction rules
1
On card hover: pencil edit icon (opacity 0→1) appears next to file name, trash/remove button appears on right. must
2
On thumbnail hover: dark overlay (rgba(0,0,0,0.5)) with photo-plus icon appears over the 88×88 thumbnail. must
3
Editing: inline text field replaces name (200×40px, border #bcc3cc), check icon to confirm, right action hidden. must
4
Uploading: text/disabled colors, spinner animation, "Uploading..." label. No actions. should
5
Error states: error row with exclamation icon. Model error also turns card border #e24f4b and status to "Failed". must
6
Card width: 779.5px in Figma, responsive in demo. Border-radius 16px, border 1px solid #d4d9e1. should
